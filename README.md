### About

Its just some test about modularizing flutter app with separate packages that force making 
app's layers independent. 

#### Workspace

Dart's workspace works well with managing dependencies within app. It cause only one `pubspec.lock` file
is being generated.

#### Build runner

Generating code with build runner also seems to not be pain anymore. Just add `--workspace` parameter
to generate code for all workspace's packages.

- for app
```bash
dart run build_runner build
```

- for workspace
```bash
dart run build_runner build --workspace 
```

## Getting Started ðŸš€

This project contains 3 flavors:

- development
- staging
- production

To run the desired flavor either use the launch configuration in VSCode/Android Studio or use the following commands:

```sh
# Development
$ flutter run --flavor development --target lib/main_development.dart

# Staging
$ flutter run --flavor staging --target lib/main_staging.dart

# Production
$ flutter run --flavor production --target lib/main_production.dart
```

_\*Monorepo Case Study works on iOS, Android, Web, and Windows._
